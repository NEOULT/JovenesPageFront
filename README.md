# JovenesPage Front (React + Vite)

Rápido scaffold con Vite + React (JS).

Comandos principales:

- Instalar dependencias:

```powershell
npm install
```

- Arrancar servidor de desarrollo:

```powershell
npm run dev
```

- Build de producción:

```powershell
npm run build
```

- Preview del build:

```powershell
npm run preview
```

URL de desarrollo por defecto: http://localhost:5173/
